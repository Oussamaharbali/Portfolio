# GPAO pour la gestion de production d'ACII

## Description du projet
Ce projet consistait à développer une application GPAO (Gestion de Production Assistée par Ordinateur) pour gérer la production d'armoires de climatisation industrielles chez CIAT. [cite: 4] L'objectif était d'optimiser la gestion des nomenclatures, des ordres de fabrication et du suivi de production.

## Mon rôle
J'ai activement participé à la conception et au développement de l'application. Mes principales tâches étaient :

*   **Conception de la base de données :** Modélisation des données et création des tables pour gérer les articles, les nomenclatures, les ordres de fabrication et les stocks.
*   **Développement de l'interface utilisateur :** Création des formulaires et des écrans pour la saisie des données, la consultation des informations et le suivi de la production.
*   **Intégration avec les systèmes existants :**  Connexion de l'application GPAO avec les systèmes de gestion de production déjà en place chez CIAT.

## Compétences utilisées
*   SQL (pour la base de données)
*   Java (pour le développement de l'application)
*   UML (pour la modélisation)

## Défis et solutions
Un défi majeur était d'intégrer l'application GPAO avec les systèmes de production existants, qui étaient hétérogènes et peu documentés. Pour surmonter ce défi, j'ai dû analyser en détail les flux de données et développer des interfaces spécifiques pour assurer la communication entre les différents systèmes.
